# Mechanical-_engineering-_week2
create a simple circuit powered by a DC motor, and write a simple algorithm that changes the speed of this motor.
## Hardware Required:
	Arduino Uno
	potentiometer
	H-bridge Motor Driver
	DC Motor
	Breadboard
	9V Battery
